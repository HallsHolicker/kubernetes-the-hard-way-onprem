# Prerequisites

# VirtualBox

Kubernetes Cluster에 필요한 컴퓨팅 리소스를 구축하기 위해 가상 머신으로 [VirtualBox](https://www.virtualbox.org/wiki/Downloads)를 사용하였습니다.

# Vagrant

VirtualBox에 이미지 프로비저닝을 쉽게 하기 위하여 [Vagrant](https://www.vagrantup.com/)를 사용하였습니다.
